# FixedGridCollectionViewLayout
Example project demoing a UICollectionViewLayout with a fixed grid layout.

Customizable with the following properties:
1. Column Count
2. Row Height
3. Item Spacing
4. Line Spacing
5. Top, Bottom, Left, Right Inset
6. Item Separators
7. Line Separators
